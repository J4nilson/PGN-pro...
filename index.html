Algoritmos

O que é

São conjuntos de passos utilizados para resolver um problema ou efetuar uma tarrefa

Exemplos

1-Levantar cedo as 6h
2-Lavar os dentes
3-Tomar banho
4-Vestir
5-Cumer
6-Apanhr o autocarro
7-Ir para Escola

Explicação

da para ver que do numero 1 até o numero 6 são passos para chegar no objetivo que é o numero 7 e como diz o conceito algoritmo são um conjunto de passos(1-6) utilizados para efetuar uma tarrefa(7)

Linguagem de programação-"C"

Quem criou

A linguagem de programação C foi criada por Dennis Ritchie. Ele desenvolveu C na década de 1970 nos Laboratórios Bell, nos Estados Unidos, enquanto trabalhava na AT&T (American Telephone & Telegraph Company). A primeira implementação de C foi em 1972, e a linguagem foi projetada para ser uma ferramenta de desenvolvimento de sistemas que operasse eficientemente em computadores e sistemas operacionais.
