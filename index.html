A linguagem de programação C foi criada por Dennis Ritchie. Ele desenvolveu C na década de 1970 nos Laboratórios Bell, nos Estados Unidos, enquanto trabalhava na AT&T (American Telephone & Telegraph Company). A primeira implementação de C foi em 1972, e a linguagem foi projetada para ser uma ferramenta de desenvolvimento de sistemas que operasse eficientemente em computadores e sistemas operacionais.

Subprogramas

Subprogramas são seções independentes de código que podem ser chamadas de diferentes partes de um programa principal para executar uma tarefa específica.

Python

Guido van Rossum criou a linguagem de programação Python em dezembro de 1989. Ele começou a trabalhar no projeto enquanto estava de férias no período natalino, com o objetivo de criar uma linguagem de programação nova que combinasse as vantagens do ABC, uma linguagem de ensino que ele ajudou a desenvolver anteriormente, com o poder e a extensibilidade de outras linguagens como C.

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral, que é amplamente reconhecida por sua simplicidade, legibilidade e flexibilidade. Foi criada por Guido van Rossum e lançada pela primeira vez em 1991. Desde então, Python se tornou uma das linguagens de programação mais populares e usadas globalmente.

VARIAVEIS

As variáveis em Python são fundamentais para a programação, permitindo armazenar e manipular dados de forma eficiente. Python suporta vários tipos de números, incluindo inteiros, ponto flutuante e números complexos.
  • variavel=input(" ") #isso retorna a um tipo string
  • variavel=str(input(" ")) # é mesma coisa do que a de cima
  • variavel=int(input(" ")) #isso pede numeros inteiros
  • variavel=float(input(" ")) #isso pede numeros reais

Entrada e saida de dados

Para ler dados do usuário, Python utiliza a função input(), que lê uma linha de texto do usuário. Por padrão, input() sempre retorna uma string. Se for necessário trabalhar com outros tipos de dados, é preciso converter a entrada usando funções de casting (int(), float(), etc.). Python oferece várias formas de exibir dados no console, sendo as mais comuns o uso das funções print() e format().

Estrutura de condição

As estruturas de condição em Python são fundamentais para controlar o fluxo de execução do programa com base em certas condições. Elas permitem que o programa tome decisões e execute blocos de código específicos dependendo de condições lógicas.
  • if(condição): #a primeira condição
  • elif(condição): #usada se hover mais condições a serem feitas el pode repitir aocontrario do if podes usala mais de uma vez
  • else: #no caso as condições a cima não forem verdadeira vai partir para o else(senão)

Estrutura de repetição

As estruturas de repetição em Python são fundamentais para executar um bloco de código múltiplas vezes, o que é essencial para tarefas que requerem iteração, como processamento de listas, loops até uma condição ser satisfeita, entre outros.
  • for Para iterar sobre uma sequência (como uma lista, tupla, string, ou um range de números).
  • while Para repetir um bloco de código enquanto uma condição é verdadeira.

Tipos de dados compostos

Em Python, os tipos de dados compostos são estruturas que permitem armazenar múltiplos valores em uma única variável. Esses tipos são essenciais para lidar com coleções de dados, como listas de itens, pares de chave-valor, e agrupamentos de elementos.
  • Listas(list) usa []
  • Tuplas(tuple) usa ()
  • Dicionario(dict) usa {}
  • Conjuntos(set) usa {} é quase a mesma coisa que o decionario

Funções

Funções em Python são blocos de código reutilizáveis que realizam uma tarefa específica. Elas permitem organizar o código em partes menores e mais gerenciáveis, facilitando a leitura, manutenção e reutilização do código. Vamos explorar os principais conceitos e características das funções em Python. A palavra-chave def, seguida pelo nome da função e parênteses (). Se houver parâmetros para a função, eles são listados entre os parênteses. O corpo da função é indentado e contém o código que define o que a função faz.
